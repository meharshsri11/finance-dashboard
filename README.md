# Finance Dashboard

## 🔗 Live Demo

👉 https://finance-dashboard-bice-mu.vercel.app/

---

## 📌 Overview

This is a responsive Finance Dashboard built using React and Tailwind CSS.
It allows users to track income, expenses, and understand their financial activity through a clean and interactive interface.

The project focuses on frontend design, component structuring, and state management using mock data.

---

## ✨ Features

### 📊 Dashboard Overview

* Summary cards: Total Balance, Income, Expenses
* Balance trend visualization (line chart)
* Spending breakdown by category (pie chart)

---

### 💳 Transactions

* View transactions with date, category, type, and amount
* Add, edit, and delete transactions (Admin role)
* Search, filter, and sort functionality

---

### 🔐 Role-Based UI

* Viewer → Read-only access
* Admin → Full access (add/edit/delete)
* Role switch using dropdown

---

### 📈 Insights

* Highest spending category
* Monthly comparison
* Basic financial insights and observations

---

## 🧠 My Approach

* Designed a clean and intuitive UI for financial tracking
* Used React Context + useReducer for state management
* Built reusable components for scalability
* Implemented role-based behavior to simulate real-world usage
* Ensured responsiveness across different screen sizes
* Added local storage for data persistence
